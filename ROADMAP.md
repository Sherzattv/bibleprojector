# Roadmap: Bible Projector 🗺️

Этот документ описывает планы по развитию приложения и созданию сопутствующих сервисов.

## 🚀 Функционал приложения (Application Features)

### 🔥 Высокий приоритет
- [ ] **Параллельный перевод**
  - Возможность выбора двух переводов для одновременного отображения на экране (Split Screen).
  - Настройка: слева основной, справа дополнительный (или сверху/снизу).
  
- [ ] **Единый поиск (Переключатель)**
  - Объединить поиск по тексту и поиск по ссылкам/книгам в один интерфейс.
  - Добавить переключатель (Toggle) или умное распознавание ввода.

- [ ] **Управление экраном трансляции**
  - Кнопки для быстрого управления вторым окном:
    - 👁️ **Открыть/Показать** (существующая)
    - 🙈 **Скрыть/Черный экран** (Blackout без закрытия окна)
    - 🖥️ **На весь экран** (Maximize/Fullscreen на втором мониторе)
    - ❌ **Закрыть** (полностью)

### 🧊 Низкий приоритет
- [ ] **Навигация по книгам (UI)**
  - Визуальное меню для выбора Книги -> Главы -> Стиха (для тех, кто не помнит сокращения).
  
- [ ] **Новые дизайны (Themes)**
  - Разработка дополнительных премиум-тем для окна трансляции.
