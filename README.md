# Вечный Свет (Eternal Light) - Система отображения стихов

Профессиональная PWA-система для вывода стихов из Библии на экраны и проекторы.

**🏠 [Посмотреть сайт проекта →](https://bibleprojector.kz/)**  
**🌐 [Открыть приложение →](https://bibleprojector.kz/app/controller.html)**

## ✨ Возможности

- 📖 **3 перевода**: Синодальный (RST), Новый Русский (NRT), Казахский (KTB)
- 🔍 **Умный поиск**: Введите `ин 3 16` или `жар 1:1` — приложение поймёт
- 📺 **Режим проектора**: Отдельное окно для вывода на второй экран
- 📝 **Быстрые заметки**: Можно транслировать произвольный текст
- 🌙 **Современный дизайн**: Glassmorphism, тёмная тема
- 📱 **PWA**: Устанавливается как приложение, работает оффлайн

## 🚀 Быстрый старт

### Вариант 1: Онлайн (рекомендуется)
1. Откройте ссылку выше в Chrome
2. Нажмите иконку "Установить" в адресной строке (⊕)
3. Приложение появится на рабочем столе и будет работать оффлайн

### Вариант 2: Локально
1. Скачайте или клонируйте репозиторий
2. Откройте `app/controller.html` в браузере
3. Нажмите "Открыть экран" для запуска окна проектора

## 🎮 Управление

| Действие | Клавиша |
|----------|---------|
| Поиск стиха | Введите ссылку + `Enter` |
| Транслировать | `Ctrl + Enter` |
| Скрыть с экрана | `Esc` |
| Полный экран проектора | `F11` |

## 📚 Форматы поиска

```
ин 3 16        → Иоанна 3:16
мф 5 3-10      → Матфея 5:3-10
1 кор 13 4     → 1-е Коринфянам 13:4
жар 1:1        → Жаратылыс 1:1 (казахский)
```
